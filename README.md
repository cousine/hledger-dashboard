# hledger Dashboard

Obsidian plugin that renders a full financial dashboard from your hledger journals.

## Setup

1. Enable the plugin in Obsidian settings.
2. Click the grid icon in the ribbon bar (or run the command `Open hledger Dashboard`).
3. If hledger isn't auto-detected via PATH, configure the path in **Settings â†’ hledger Dashboard**.

## Required Journal Files

The plugin reads from `finances/finances.journal` (configurable). For full functionality, these companion files should exist:

| File | Purpose | Included in `finances.journal`? |
|---|---|---|
| `finances/prices.journal` | Market prices for USDâ†’EGP conversion (`P 2026-01-01 $ EGP 50`) | Yes |
| `finances/2026-budget.journal` | Monthly budget periodic transactions | Yes |
| `finances/2026.journal` | Main transaction data (auto-generated by `hledger import`) | Yes |

## Dashboard Panels

| Panel | Description | Data Source |
|---|---|---|
| **Net Worth** | KPI cards + balance sheet table (EGP / USD / converted) | `balance assets liabilities -H` |
| **Cash Flow** | Income vs expenses for the period + bar chart | `balance income expenses -p PERIOD` |
| **Expense Breakdown** | Treemap chart of expense categories | `balance expenses: --tree -p PERIOD` |
| **Credit Cards** | Credit card balances | `balance liabilities:creditcard: -H` |
| **Loan (CIB)** | Loan balance, interest, fees paid | `balance + register` |
| **Budget vs Actual** | Per-category budget tracking with progress bars | `balance --budget -p PERIOD` |
| **Uncategorized** | Flagged transactions in `equity:uncategorized` | `register equity:uncategorized` |
| **Recent Transactions** | Last N register entries in the period | `register -n -p PERIOD` |

## Period Controls

Toggle between **Month**, **Quarter**, and **YTD** views using the toolbar buttons. Use the month picker to select the reference month.

## Currency Display

- Accounts shown in their native currency (EGP / USD columns)
- Converted totals use market prices from `prices.journal` via `hledger -X EGP`

## Budget Setup

Create a `finances/2026-budget.journal` file with monthly periodic transactions:

```journal
~ monthly from 2026-01-01
    (expenses:essentials:mivida)          EGP 20000
    (expenses:non-essentials:leisure)     EGP 5000
```

Then add `include 2026-budget.journal` to `finances.journal`.

## Settings

| Setting | Description |
|---|---|
| hledger binary path | Path to hledger (default: "hledger" via PATH) |
| Journal file | Relative path from vault root to your .journal file |
| Recent transactions | Number of transactions in the recent panel |
| Default period | Default view on open (Month / Quarter / YTD) |

## Requirements

- Obsidian Desktop (requires `child_process` / shell access)
- [hledger](https://hledger.org) 1.52+ installed and in PATH
